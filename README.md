# BBQ Wood Supply - Single Page Website

A modern, responsive single-page website for **BBQ Wood Supply**, built with **HTML, CSS, and TypeScript**. Features smooth scrolling, dynamic hero, product showcase, and contact section. Fully mobile-first and production-ready.  

---

## Features

- **Dynamic Hero Section**: Eye-catching welcome with full-width background.  
- **Our Mission Section**: Clean, minimal layout highlighting company values.  
- **Products Section**: Sliding product showcase with modern design.  
- **Reach Out Section**: Friendly contact area (no form) with CTA buttons.  
- **Smooth Scroll & Scroll Animations**: Fade-in/out and offset scrolling.  
- **Responsive Design**: Mobile-first, scales to larger screens with media queries.  
- **CSS Variables**: Easy theme customization (`--primary-color`).  
- **Production Build**: `/dist` folder with compiled JS, CSS, and assets.
